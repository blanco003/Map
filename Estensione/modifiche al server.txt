Modifiche al sever rispetto alla versione base: 

TableData :
- aggiunta nomi_tabelle_presenti() per recuperare i nomi di tutte le tabelle presenti sul db.
- aggiunta crea_nuova_tabella(table,num_esempi) che crea una nuova tabella sul db.
- aggiunta inserisci_valori(table,valori) che inserisce la tupla nella tabella sul db.
- aggiunta elimina_tabella(table) che elimina la tabella dal db

ServerOneClient :
- il client invia 0 per caricare un dataset gia esistente, 1 per creare un nuovo dataset (tabella sul db), 5 per eliminare una tabella dal db
- quando il client durante il caricamento da file inserisce un file inesistente o un file in cui non è salvato un clustering, anziché terminare la comunicazione con il client come nella versione base viene richiesto al client di inserirne uno valido,
- quando il client durante l'apprendimento da db inserisce una profondita che supera il numero di esempi nel dataset o negativa o zero, anziché terminare la comunicazione con il client come nella versione base viene richiesto al client di inserire una profondita valida
- il client ha la possibilità di eliminare un dataset
